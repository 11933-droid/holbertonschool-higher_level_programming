# Python - Server Side Rendering

This project demonstrates server-side rendering (SSR) techniques using Python and Flask framework with Jinja2 templating engine.

## Tasks

### Task 0: Creating a Simple Templating Program
Python function that generates personalized invitation files from a template with placeholders.

### Task 1: Creating a Basic HTML Template in Flask
Basic Flask application serving web pages with reusable header and footer templates.

### Task 2: Creating a Dynamic Template with Loops and Conditions
Flask application with dynamic content rendering using Jinja loops and conditionals.

### Task 3: Displaying Data from JSON or CSV Files
Flask application that reads and displays product data from JSON and CSV files.

### Task 4: Extending Dynamic Data Display to Include SQLite
Extended Flask application with SQLite database support for product data.

## Requirements

- Python 3.x
- Flask
- SQLite3

## Installation

```bash
pip install Flask
```

## Usage

Run each task file:
```bash
python3 task_01_jinja.py
python3 task_02_logic.py
python3 task_03_files.py
python3 task_04_db.py
```

Access the application at `http://localhost:5000`
