# JavaScript Warm Up

This project is part of the Holberton School Higher Level Programming curriculum.  
It introduces the basics of JavaScript using Node.js through simple scripting tasks.

## Description

The goal of this project is to get familiar with:
- Running JavaScript scripts with Node.js
- Variables (`const`, `let`)
- Command-line arguments
- Control flow (if, loops)
- Functions
- Arrays and objects
- Recursion
- Basic module exports

Each task focuses on a specific JavaScript concept.
